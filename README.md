# NIELIT Community Platform

A **self-hosted, free, open-source** community platform combining Facebook + WhatsApp features.

## Features

✅ **Activity Feed** — Facebook-style news feed  
✅ **Groups** — Create and join communities  
✅ **Hashtag Topics** — Follow #topics  
✅ **Events** — Create events and RSVP  
✅ **Audio Calls** — One-click like WhatsApp  
✅ **Video Calls** — HD video with ringing  
✅ **Screen Share** — Share your screen like Zoom  

## Everything Included & Pre-Configured

This repository has **everything ready to run**:

- ✅ WordPress with all plugins installed
- ✅ BuddyPress (social features)
- ✅ bbPress (forums)
- ✅ The Events Calendar
- ✅ Audio/Video calling system (LiveKit)
- ✅ Database configuration
- ✅ LiveKit API configuration
- ✅ Web server configuration

## Deploy in 3 steps

1. **Create Ubuntu 24.04 VM** on Azure/AWS/GCP (~$30/month)
2. **Point domain** DNS to VM IP
3. **Clone this repo and follow** installation steps

## Cost

- VM: ~$30/month
- Domain: ~$12/year
- Software: $0 (everything free)
- **Total: ~$30/month**

## System Requirements

- Ubuntu 24.04 LTS
- 2+ vCPU, 4+ GB RAM
- 50 GB storage
- Own domain
